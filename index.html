Boot to Gecko

Desenvolvido pela Mozilla. O Boot to Gecko (B2G) é um projeto de sistema operacional móvel. Fazendo uso de um Kernel Linux e permitindo que os usuários utilizem apps inteiramente desenvolvidos utilizando padrões da Web Aberta.

Nome comercial do B2G, o Firefox OS traz a essência da Mozilla. Sendo aberto e acessível a todos.
